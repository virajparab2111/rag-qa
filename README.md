# RAG-based Policy Question Answering System

This project implements a Retrieval-Augmented Generation (RAG) pipeline for answering policy-related questions using structured documents and PDFs.

## Features
- ChromaDB vector storage
- Cross-encoder reranking
- Phi-3 Mini (Ollama) LLM
- JSON-constrained outputs
- Source citations per answer
- Confidence scoring
- Automatic evaluation

## Tech Stack
- Python
- LangChain
- ChromaDB
- SentenceTransformers
- Ollama (Phi-3 Mini)
